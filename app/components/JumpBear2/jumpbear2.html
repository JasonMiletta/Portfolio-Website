JumpBear 2


JumpBear 2 is a work-in-progress sequel to my first proper game, JumpBear. The game is a 3D action platformer, in similar style of older xbox/ps2 era collect-a-thon games. All of the assets are created by myself using the following tools:

Blender for modeling, rigging, and animations

Krita for any texture work

Ableton for any audio assets

Unreal Engine 5
